# Html-interactive-movie-guide
A coding challenge using raw HTML consisting on a web page with a simple movie guide!

## Objective ->

Create an HTML page that works as a mini catalog of fictional movies, using structural and multimedia elements.

## Mandatory Requirements ->

### Header:
Add the website name, for example: “CineTop” or any fun title.
Include a navigation menu linking to the page sections.

### “Featured Movies” Section:
#### Display 3 movies containing:
  * Movie title;
  * Short description;
  * Movie poster image;
  * “Learn more” link.

### “Genres” Section: 
Create a list of movie genres.
Each genre can contain a nested list with 2 or 3 movie titles.

### “Fun Facts” Section: 
Use quotation elements to display fun facts about cinema.

### Footer:
Add copyright text or a fun movie-related phrase.

## Extra Challenges ->
Use internal links to navigate between sections.
Use alt attributes correctly in images.
Research how to embed YouTube videos.
Try creating nested lists inside the genres section.
